## FarmPal 👨‍🌾
경기대학교 심화캡스톤 프로젝트 FarmPal
&nbsp;

&nbsp;

### 프로젝트 설명 📖
> 초보 농부를 포함한 모든 농민들에게 도움이 되고자 하는 마음에서 시작한 프로젝트입니다.
&nbsp;

&nbsp;

### 기능 구성
 - AI 병충해 판별 🔎
   - AI를 활용해 촬영한 작물의 감염 질병 여부 및 종류 판별
     
 - 직거래 장터 🚚
   - 재배한 농작물을 쉽게 거래 가능
     
 - 농촌 체험 모집 / 신청 🌱
   - 농민들이 체험 프로그램을 직접 개설하여 모집
     
 - 커뮤니티 (Q&A / 자유게시판) 👩🏼‍🤝‍🧑🏻
   - 농민 간 지식 및 경험 공유를 위한 커뮤니티 개설
